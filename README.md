## ğŸ”¬ Instruction

Simple and high settable table for all devices, CSS only; create your component with a super tiny code

Import with

```js
import { ResponsiveTable } from "responsive-table-react";
```

### Const example

```ts
const columns = [
    {
      "field": "name",
      "label": "Name"
    },
    {
      "field": "surname",
      "label": "Surname"
    }
  ]

  const data = [
    {
      "name": "Mark",
      "surname": "Garsin"
    },
    {
      "name": "Gabriel",
      "surname": "Betappi"
    },
    {
      "name": "Gustav",
      "surname": "Mahler",
    }
  ]

  return (
    <ResponsiveTable columns={columns} data={data} />
  )
```

## Props

Set the designOptions

- `bordered?: boolean` <br/>
Add the border

- `dark?: boolean` <br/>
Add the dark mode 

- `color?: string` <br/>
Set the master color
