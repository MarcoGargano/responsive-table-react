🔧 Installation

Install by NPM

npm i responsive-table-react

Here you can find the package on NPM: responsive-table-react.

🔬 Instruction

Simple and high settable table for all devices, CSS only; create your component with a super tiny code.

Import with:

import { ResponsiveTable } from "responsive-table-react";

Example

const columns = [
    {
      "id": "name",
      "text": "Name"
    },
    {
      "id": "surname",
      "text": "Surname"
    }
  ]

  const data = [
    {
      "name": "Mark",
      "surname": "Garsin"
    },
    {
      "name": "Gabriel",
      "surname": "Betappi"
    },
    {
      "name": "Gustav",
      "surname": "Mahler",
    }
  ]

  return (
    <ResponsiveTable columns={columns} data={data} />
  )

🌊 Props

designOptions

  • Add the border: bordered?: boolean // es: bordered?: true
  • Add the dark mode: dark?: boolean // es: dark?: true
  • Set the master color: color?: string // es: color?: #CC0000
